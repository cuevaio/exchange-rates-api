## Implementation

1. Change the upstream URL to `https://cuantoestaeldolar.pe/cambio-de-dolar-online`.
2. Replace the current `__NEXT_DATA__` extraction with a small same-file parser that:
   - finds one or more `self.__next_f.push([1, "..."])` script chunks in the HTML
   - safely decodes the escaped string payload with string parsing and `JSON.parse` only
   - locates the `exchangeHouses` array content inside the decoded payload
   - parses out each house's `title`, `site`, `bank`, `rates.buy.cost`, and `rates.sale.cost`
3. Normalize parsed values:
   - convert buy/sell rates to numbers
   - keep the existing website cleanup replacements
   - split `bank` into a trimmed array when present
4. Filter out invalid houses before computing aggregates:
   - missing or empty `title`
   - missing, non-numeric, or non-positive buy/sell rates
5. Keep the response shape unchanged:
   - `updatedAt`
   - `best.maxBuy`
   - `best.minSell`
   - `meta.buyAvg`
   - `meta.sellAvg`
   - `meta.count`
   - `houses`
6. Add defensive error handling for:
   - missing script payload
   - failed payload decoding
   - empty valid-house result set
